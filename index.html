Alfa Projekt

Czym jest Angular?

  • framework do budowania aplikacji zwanych SPA (Single Page Application)
  • pozwala na użycie HTML'a jako języka szablonów
  • jest komplentnym rozwiązaniem ale tylko po stronie klienta
  • zaprojektowany z myślą o aplikacjach typu CRUD (Create Read Update Delete)
  • nakierowany na testy jednostkowe i testy E2E

Zalety i wady

  • implementacja MVC
  • szybki dewelopment - “mniej kodu mniej błędów”
  • dwukierunkowe bindowanie
  • rozszera HTML
  • łatwo testowalny
  • stroma krzywa poznawcza
  • złożonośc koncepcji
  • “$scope soup” i pisanie w Angularze
  • słabo rozbudowana dokumentacja
  • wydajność
  • problemy z SEO

Expression
Wyrażenie — zapis pozwalający na dostęp do zmiennych z kontekstu
Compiler
Kompilator szablonów wraz z dyrektywami
Filter
Filtr formatujący wartość wyrażenia
View
Widok — to co widzi uzytkownik DOM

Pojęcia i koncepcje

Data Binding
Bindowanie — synchronizacja danych pomiędzy modelem i widokiem
Controller
Kontroller — logika stojąca za widokiem
Module
Moduł łączący części aplikacji np. filtry, dyrektywy, serwisy
Service
Serwis — osobny komponent, klasa niezależna od widoku

Moduły

  • ngAnimate — animacja z użyciem jQuery i CSS
  • ngAria — wsparcie dla WAI-ARIA (Accessible Rich Internet Applications)
  • ngResource — wsparcie do pracy z RESTful API
  • ngTouch — wsparcie dla urządzeń mobilnych
  • ngRoute — wparcie dla routingu

Serwisy (dostarczane przez moduły)

  • $http — dostarcza funkcjonalności do komunikacji via HTTP
  • $interval — wrapper na window.setInterval
  • $log — prosty serwis logowania bazujący na obiekcie console
  • $window — referencja do obiektu window
  • $q — pozwala uruchamiać funkcje asynchronicznie (implementacja obietnicy)

Serwisy

app.js

var myApp2 = angular.module('myServiceModule', []);

myApp2.controller('MyController', ['$scope', 'notify', function ($scope, notify) {
        $scope.callNotify = function (msg) {
            notify(msg);
        };
    }]);
myApp2.factory('notify', ['$window', function ($window) {
        var msgs = [];
        return function (msg) {
            msgs.push(msg);
            if (msgs.length === 3) {
                $window.alert(msgs.join("\n"));
                msgs = [];
            }
        };
    }]);

Konteksty — $rootScope,$scope

app.js

var myApp3 = angular.module('myScopeModule', []);
myApp3.controller('AController', ['$rootScope', '$scope', function ($rootScope, $scope) {
        $scope.message = '';
        $scope.broadcast = function () {
            $rootScope.$broadcast('event-a', 'A');
        };
        $scope.$on('event-b', function (event, args) {
            $scope.message = $scope.message.concat(args);
        });
    }]);
myApp3.controller('BController', ['$rootScope', '$scope', function ($rootScope, $scope) {
        $scope.message = '';
        $scope.broadcast = function () {
            $rootScope.$broadcast('event-b', 'B');
        };
        $scope.$on('event-a', function (event, args) {
            $scope.message = $scope.message.concat(args);
        });
    }]);

    <button ng-click="broadcast();">Nadawaj B</button>

Dyrektywy

app.js

var myApp6 = angular.module('myDirectiveModule', []);

myApp6.directive('customCheckbox', function () {
    return {
        restrict: 'E',
        replace: true,
        transclude: true,
        template: '<span class="checkbox" ng-class="{checked: isChecked}" ng-click="toggleMe()"></span>',
        scope: {isChecked: '=?'},
        link: function (scope, elem, attrs) {
            scope.isChecked = false;
            scope.toggleMe = function () {
                scope.isChecked = !(scope.isChecked);
            }
        }
    }
});
